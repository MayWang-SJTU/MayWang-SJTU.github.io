Indoor Localization

This work mainly focus on studying the temporal correlation effects on RSS fingerprinting based indoor localization method. From my theoretical derivation and scientific calculation, we found that temporal correlation of signals can definitely correct the localization criteria for MLE method, so that it can significantly improve the fundamental limits like accuracy and reliability. With expected experiment results, I have wrote a paper as the first author and submitted it for INFOCOM this summer.

Cellular Network

During the second half of last year, I assisted in the completion of a cooperation project between IWCT (Institute of Wireless Communication and Technology) and Ericsson Corp. I renovate the traffic model and wrote a simulation software by C++ to model the user behavior propagation in 3GPP communication network. The project is finally well recognized. In addition, this project gave me an idea on NFV network optimization, which is one of my leading ongoing research project.

Intelligent Transportation

We build a crowdsourcing based lane localization system to identify the number of lanes in a road and locate which lane you are driving at in real time, so that it can facilitate the travel programming of the pilotless automobile and high precision vehicle navigation. The key idea of the system
is leveraging the drivers’ smartphones to collect information from both embedded GPS module and sensors, which are then jointly processed with a machine learning mechanism.
